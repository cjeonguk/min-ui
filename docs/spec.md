# 교내 커뮤니티 웹 앱 기능 명세서

---

## 1. 프로젝트 개요

**목표** 교내 학생들의 공지 및 소통을 위한 커뮤니티 웹 애플리케이션

**레퍼런스** Facebook / Instagram (경로의존성을 고려해 유사한 UI·UX 구조 채택)

**지원 환경** PWA(Progressive Web App) — 모바일 사용 최적화

---

## 2. 레이아웃 구조

### 데스크톱

| 영역 | 내용                      |
| ---- | ------------------------- |
| 좌측 | 메뉴 바                   |
| 중앙 | 포스트 / 그룹 / 페이지 등 |
| 우측 | 알림                      |

### 모바일

| 영역      | 내용                    |
| --------- | ----------------------- |
| 상단 버튼 | 알림 (별도 탭으로 분리) |
| 하단      | 메뉴 바                 |

---

## 3. 메뉴 바 구성

메인 / 검색 / 그룹 / 채팅 / 마이페이지

---

## 4. 주요 기능

### 4-1. 메인 (랜딩 / 대시보드)

- **비로그인 상태** 랜딩 겸 로그인 페이지 표시
- **로그인 상태** 대시보드 표시
  - 최근 포스트 피드
  - 스토리 목록

---

### 4-2. 회원가입 및 인증

1. 구글 계정으로 소셜 로그인
2. 로그인 후 학생 정보 추가 입력
3. 관리자 승인 대기 상태로 전환

**대기 페이지**

회원가입은 완료되었으나 재학생 여부가 미확인된 계정이 관리자 승인을 기다리는 전용 페이지

---

### 4-3. 그룹

Facebook 그룹과 유사한 구조

- 전체 공지, 학생 개인 공지 등 다양한 종류의 그룹 존재
- 포스트는 그룹 단위로 게시
- 특정 그룹은 권한이 부여된 사용자만 포스트 작성 가능 (권한 제어)

---

### 4-4. 페이지 (프로필)

- 개인 프로필 페이지
- 개인 포스트 작성 및 조회
- 스토리 작성
- 생일 표시

---

### 4-5. DM (다이렉트 메시지)

- 1:1 개인 채팅
- 그룹 채팅
- 파일 전송 / 이미지 전송 / 링크 전송
- E2E(End-to-End) 암호화 지원

---

### 4-6. 알림

- 댓글 알림
- 좋아요 알림
- 기타 상호작용 알림

---

## 5. 부가 기능

메인 페이지 상단 버튼으로 접근

### 5-1. 시간표

- 수강 신청한 강의 목록을 입력하면 자동으로 시간표 생성

### 5-2. 예약

#### 노래방 예약

- 원하는 사용 시간대를 사전 예약 신청
- 예약 슬롯은 매주 초기화

#### 공강(빈 강의실) 예약

- 개인 활동 등을 위한 빈 강의실 사용 시간 사전 예약
- 예약 슬롯은 매주 초기화
